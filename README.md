# Automated Project Setup

This project is a Bash script that automatically creates a project folder structure for an attendance system.

It creates the main directory, subfolders, and required files, then checks if everything was created correctly. It also checks if Python3 is installed on the system.

## What it does
- Creates a project folder
- Creates Helpers and reports folders
- Creates required files
- Checks if Python3 is installed
- Validates the project structure

## How to use
Run the script:

```bash
chmod +x setup_project.sh
./setup_project.sh
